# Introduction

Ma Radio Groove API enables third-party platforms — such as private party planning apps — to interact with the Ma Radio Groove streaming service by integrating curated DJ mixes into their digital experiences.

## Prerequisites

This tutorial assumes you have:

* Created a Ma Radio Groove streaming account.
* Generated an API key from the Ma Radio Groove Developer Dashboard.
* Installed the necessary tool to send HTTP requests (e.g. cURL, Postman).
