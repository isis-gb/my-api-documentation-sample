# Getting Started with the Ma Radio Groove API

## About this guide

This guide provides a step-by-step overview of how to integrate the Ma Radio Groove API from the Ma Radio Groove streaming service, into third-party apps.
It introduces API reference and endpoints, includes a practical use case example, and presents recommended best practices to ensure a smooth integration process.

The documentation is structured  to help you:

* Authenticate with the API.
* Retrieve DJs and curated mixes.
* Access detailed mix metadata.
* Configure third-party apps to stream DJ content.

!!! note
    All examples use fictional data for demonstration purposes.
